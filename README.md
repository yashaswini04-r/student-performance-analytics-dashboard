# Student Performance Analytics Dashboard

## Project Overview

The Student Performance Analytics Dashboard is a web application developed using Python, Flask, SQLite, HTML, CSS, and Chart.js.

This project helps manage student records and analyze their academic performance through an interactive dashboard.

---

## Features

- Add Student Records
- Search Students
- Edit Student Information
- Delete Student Records
- View Student Performance Data
- Total Students Analytics
- Average Marks Calculation
- Average Attendance Calculation
- Top Performers Analytics
- Marks Visualization using Charts
- Attendance Visualization using Charts
- SQLite Database Integration

| name | TEXT |
